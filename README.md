## 功能特性

- **主屏幕**: 安卓风格的应用网格布局
- **状态栏**: 显示时间、信号、WiFi和电池状态
- **动态壁纸**: 集成Bing每日壁纸，自动更新精美壁纸
- **明亮主题**: 清新明亮的设计风格，提供舒适的视觉体验
- **大号时间显示**: 24小时制时间显示，实时同步系统时间
- **电池状态**: 实时显示设备电量和充电状态
- **搜索功能**: 快速搜索应用、联系人和设置
- **触摸交互**: 支持点击、长按和滑动手势
- **响应式设计**: 适配不同屏幕尺寸和方向

## 技术实现

- **HTML5**: 语义化标签和现代Web标准
- **CSS3**: Flexbox和Grid布局，CSS动画和过渡效果
- **JavaScript**: 原生JS实现交互功能
- **PWA**: 渐进式Web应用，支持离线访问和安装到主屏幕

## 使用说明

1. 直接在浏览器中打开 `index.html` 文件
2. 在支持的浏览器中，可以将应用添加到主屏幕
3. 点击应用图标可以打开对应的应用界面
4. 使用底部导航栏切换不同功能页面
5. 在搜索框中输入内容可以搜索应用、联系人和设置
6. 点击快捷设置可以快速切换系统设置
